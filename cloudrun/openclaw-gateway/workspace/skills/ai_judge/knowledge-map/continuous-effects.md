# 持续性效应复杂场景

> 规则章节: 613.x, 604.3, 611.x

## 层系统深入回顾

**613.1** 持续性效应在层系统中按顺序应用：

| 层 | 内容 | 副层 |
|----|------|------|
| 1 | 复制效应 | - |
| 2 | 改变操控权 | - |
| 3 | 改变文本 | - |
| 4 | 改变类别 | - |
| 5 | 改变颜色 | - |
| 6 | 添加/移除异能 | - |
| 7 | 改变力量/防御力 | a-d |

## 关键规则再确认

### 613.6: 跨层效应

**各部分独立生效**

即使持续性效应的一部分（层6：异能）被移除，其他层的效果（层4：成为生物、层7b：P/T）仍然保留。

**经典案例：魁渡 + 潮缚师**
- 魁渡效应：层4（成为生物）+ 层6（辟邪）+ 层7b（3/4）
- 潮缚师移除层6的异能
- 结果：魁渡是 3/4 生物鹏洛客，无辟邪、无忠诚异能

### 613.8: 从属关系

**必须在同一层或副层才存在从属关系**

不同层的效应**永远不存在从属关系**，适用613.6跨层规则。

## 复杂场景分析

### 场景1：多个「成为」效应

**案例**：一个生物被两个效应影响：
- 效应A：「成为蓝色」
- 效应B：「成为黑色」

**分析**：
- 两者都在**层5**（改变颜色）
- 无时戳依赖 → 按时戳顺序应用
- 后应用的生效

### 场景2：「成为」与「失去」

**案例**：
- 效应A：「成为神器生物」
- 效应B：「失去所有类别」

**分析**：
- 两者都在**层4**（类别改变）
- 无时戳依赖
- 按时戳顺序应用

### 场景3：文本改变 + 异能添加

**案例**：
- 效应A（层3）：「失去所有异能」（文本改变）
- 效应B（层6）：「获得辟邪」（添加异能）

**分析**：
- 不同层！
- 先应用层3：失去印刷异能
- 再应用层6：获得辟邪
- 结果：具有辟邪

### 场景4：P/T设定的复杂互动

**层7副层**：
- 7a: 定义P/T的特征定义异能（CDA）
- 7b: 设定P/T的效应
- 7c: 改变P/T但不设定的效应（+1/+1指示物在此层后应用）
- 7d: 交换P/T的效应

**案例**：
- 「变成3/3」vs「+3/+3」vs「*/*等于墓地生物数」

## 时戳规则 (613.7)

### 613.7a: 时戳的获得

- **永久物**：进入战场时获得时戳
- **贴附的灵气/武具**：贴附时获得新时戳
- **其他持续性效应**：效应产生时获得时戳

### 613.7b: 时戳更新

- **贴附的灵气/武具**：贴附到新永久物时，获得新时戳
- **改变贴附物**：改变贴附物时获得新时戳

### 613.7c: 从属关系决定时戳顺序

如果确定存在从属关系，有依赖的效应**后应用**（不考虑时戳）。

## 特征定义异能 (CDA)

**604.3** 特征定义异能定义物体的特征，格式如：
- 「此生物的力量和防御力各等同于[某数值]」
- 「此生物的颜色为[某颜色/颜色集合]」

### CDA 的关键特性

1. **604.3a**: CDA 在**所有区域**生效（除非该异能只在特定区域有意义）
2. **604.3b**: CDA 在**层7a**生效（对于P/T定义）
3. **604.3c**: CDA 不是**文本改变效应**，即使它定义了特征

### CDA 案例

**多明尼亚的虚影**（Tarmogoyf）
- 「*/* +1」等于「所有坟墓场中牌张类别数量 +1」
- 这是CDA，在层7a计算
- **陷阱**：这不是「文本改变」，所以「失去所有异能」不影响CDA的运作！

**纠正**：
- 如果生物「失去所有异能」，它失去CDA
- 如果没有CDA定义P/T，则使用印刷的P/T（或0/0）

## 「成为」效应详解

### 611.3: 持续性效应的持续时间

「成为」效应通常持续到「回合结束」或永久。

### 611.3a: 一些效应的持续时间会改变

如「只要此生物为已横置」表示条件性持续。

### 611.3b: 效应的组成部分

一个持续性效应可能包含多个组成部分，分别影响不同层。

**案例**：「成为3/4，获得辟邪」
- 层7b：P/T设定为3/4
- 层6：获得辟邪异能
- 可能还有其他层的效果

## 复杂综合案例

### 案例：魁渡 + 史芬斯的训谕 + 潮缚师

**场况**：
1. 魁渡进场，成为3/4生物鹏洛客并具有辟邪
2. 史芬斯的训谕：「所有生物失去所有异能」
3. 潮缚师：「目标生物失去所有异能」

**分析**：
- 魁渡的效应在多层：4/6/7b
- 史芬斯的训谕（层3/6）：使生物失去异能
- 潮缚师（层6）：移除异能

**结果**：
- 层4：魁渡仍然是生物
- 层6：失去所有异能（辟邪、忠诚异能）
- 层7b：仍然是3/4

### 案例：腥红之月 + 乌尔博格 + 液金包覆

**场况**：
- 腥红之月：所有非基本地都是山脉
- 乌尔博格：所有地都是沼泽，具有「横置：加B」
- 液金包覆：目标永久物成为4/4神器生物

**分析**：
- 腥红之月和乌尔博格都在**层4**（类别改变）
- 存在从属关系：乌尔博格依赖腥红之月
- 液金包覆在层4和层7b

**结果**：
- 层4：腥红之月先应用（所有非基本地=山脉），乌尔博格后应用
- 层7b：液金包覆设定P/T为4/4

## 常见误区总结

1. **「失去异能」≠ 「不再是生物」**：
   - 层6和层4是独立的

2. **CDA 不是文本改变**：
   - 但「失去异能」仍然会影响CDA（因为CDA是异能）

3. **时戳不是万能的**：
   - 从属关系会覆盖时戳顺序

4. **不同层永远没有从属**：
   - 牢记：同层才从属，跨层各生效
