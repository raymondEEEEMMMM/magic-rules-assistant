## 如何查找规则

### 方法1: 按规则号查找
规则格式为 `XXX.Y` 或 `XXX.Ya`，如：
- `100.1` → 第1章，100节，第1条
- `702.19` → 第7章，关键字异能，践踏
- `701.2` → 第7章，关键字动作，起动

### 方法2: 按关键词查找
所有关键词都在 **7.md** 中：
- 701.x = 关键字动作 (施放、牺牲、占卜等)
- 702.x = 关键字异能 (飞行、死触、连击等)

### 方法3: 查词汇表
- **glossary.md** = 英文首字母排序
- **glossarycn.md** = 拼音首字母排序

## 常用规则速查

### 回合结构 (5.md)
```
开始阶段
  ├─ 重置步骤 (#cr502)
  ├─ 维持步骤 (#cr503) ← "在维持开始时"触发
  └─ 抓牌步骤 (#cr504)

行动阶段 (#cr505) ← 可以使用地、施放咒语

战斗阶段 (#cr506)
  ├─ 战斗开始步骤 (#cr507)
  ├─ 宣告攻击者步骤 (#cr508)
  ├─ 宣告阻挡者步骤 (#cr509)
  ├─ 战斗伤害步骤 (#cr510)
  └─ 战斗结束步骤 (#cr511)

终结阶段
  ├─ 结束步骤 (#cr513) ← "在结束步骤开始时"触发
  └─ 清除步骤 (#cr514) ← 清除伤害、检查状态动作
```

### 优先权 (117)
- 有优先权的牌手可以施放咒语、起动异能
- 主动牌手先获得优先权，然后非主动牌手依次获得

### 层系统 (613) - 持续性效应生效顺序
```
1. 复制效应
2. 改变操控权效应
3. 改变文字栏效应
4. 改变类别、副类别、超类别效应
5. 改变颜色效应
6. 添加或移除异能效应
7. 添加或移除传奇/基本超类别、改变指示物效应
8. 改变力量/防御力特征定义异能
9. 其他改变力量/防御力效应 (按时间印)
```

## 状态动作 (704)

状态动作自动执行，不进入堆叠：

| 动作 | 触发条件 |
|------|----------|
| 生物死亡 |  toughness≤0 或受到致命伤害 |
| 传奇规则 | 操控多个同名传奇永久物 |
| 鹏洛客 uniqueness | 操控多个同名鹏洛客 |
| 生物离场 | 被 attached 的灵气和武具置入坟墓场 |
| 灵气检查 | 所结附的物件不合法时置入坟墓场 |
