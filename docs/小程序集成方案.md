## API 接口

### 1. 规则搜索

**端点**: `GET /api/search`

**参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| q | string | 是 | 搜索关键词 |

**响应**:
```json
{
  "query": "战斗",
  "count": 5,
  "results": {
    "rules": [
      {
        "rule_number": "501.1",
        "title": "战斗开始",
        "content": "..."
      }
    ]
  }
}
```

### 2. 关键词查询

**端点**: `GET /api/keyword`

**参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| k | string | 是 | 关键词 |

**响应**:
```json
{
  "keyword": "Flying",
  "result": {
    "keyword_name": "Flying",
    "description": "具有飞行异能的生物不能被不具有飞行异能的生物阻挡。",
    "full_text": "702.9. 飞行...",
    "examples": [...]
  }
}
```

### 3. 卡牌查询

**端点**: `GET /api/card`

**参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| q | string | 是 | 卡牌名称 |
| page | int | 否 | 页码 |
| page_size | int | 否 | 每页数量 |

**响应**:
```json
{
  "items": [
    {
      "name": "闪电击",
      "name_en": "Lightning Bolt",
      "mana_cost": "{1}{R}",
      "type_line": "瞬间",
      "oracle_text": "闪电击对任意一个目标造成3点伤害。",
      "rarity": "U",
      "set_code": "M19",
      "collector_number": "146"
    }
  ],
  "total": 100,
  "page": 1,
  "page_size": 20
}
```

### 4. 随机卡牌

**端点**: `GET /api/random`

**响应**:
```json
{
  "result": {
    "name": "闪电击",
    "mana_cost": "{1}{R}",
    "type_line": "瞬间",
    "oracle_text": "..."
  }
}
```

## 小程序页面设计

### 1. 首页 (index)

- **Banner区**: 展示服务名称和Slogan
- **功能入口**: 4个快捷入口卡片
  - 关键词查询
  - 规则搜索
  - 卡牌查询
  - 综合搜索
- **热门关键词**: 可点击的关键词标签
- **今日随机卡牌**: 点击展示随机卡牌

### 2. 搜索页 (search)

- **搜索框**: 支持搜索规则/关键词/卡牌
- **类型筛选**: 全部 / 规则 / 关键词 / 卡牌
- **搜索历史**: 记录最近搜索
- **结果展示**: 分类型展示搜索结果

### 3. 卡牌页 (card)

- **搜索框**: 输入卡牌名称
- **卡牌列表**: 图片+名称+类型+稀有度
- **详情弹窗**: 完整卡牌信息

### 4. 关键词页 (keyword)

- **搜索框**: 输入关键词
- **结果展示**: 关键词解释+完整规则+示例

### 5. 规则页 (rule)

- **搜索框**: 输入规则关键词
- **结果展示**: 规则编号+标题+内容

## 云开发配置

### 1. 创建云环境

在 CloudBase 控制台创建云环境：

```
环境名称: magic-rules-assistant
环境ID: magic-rules-assistant-xxx
```

### 2. 部署云函数

使用 `cloudbaserc.json` 配置：

```json
{
  "envId": "magic-rules-assistant-xxx",
  "functionRoot": "functions",
  "functions": [
    {
      "name": "magic-rules-api",
      "config": {
        "timeout": 60,
        "envVariables": {}
      }
    }
  ]
}
```

### 3. 配置小程序

在 `app.js` 中初始化：

```javascript
wx.cloud.init({
  env: 'magic-rules-assistant-xxx',
  traceUser: true
})
```
