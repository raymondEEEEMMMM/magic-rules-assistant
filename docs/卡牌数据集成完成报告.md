### 2. 卡牌查询服务（CardService）

**文件**: `backend/services/card_service.py`

**功能**:
- ✅ 按名称搜索卡牌（模糊匹配）
- ✅ 按 UUID 获取卡牌
- ✅ 按关键词异能搜索卡牌
- ✅ 获取关键词异能列表
- ✅ 获取数据库统计信息
- ✅ 获取向量化数据

**API 方法**:

```python
from services import CardService

service = CardService()

# 按名称搜索
cards = service.search_cards_by_name("Black Lotus", limit=5)

# 按关键词搜索
cards = service.get_cards_by_keywords(["Flying", "Trample"], limit=10)

# 获取统计信息
stats = service.get_statistics()
# {"total_cards": 109030, "total_sets": 749, ...}

# 获取关键词列表
keywords = service.get_keywords_list()
# ["Flying", "Trample", "First Strike", ...]

# 获取向量化数据
cards = service.get_card_text_for_vectorization(limit=100)
```

### 集成到规则系统

```python
from services import RuleService, CardService

# 扩展 RuleService
class RuleService:
    def __init__(self, db):
        self.db = db
        self.card_service = CardService()

    def search_with_cards(self, query: str):
        """同时搜索规则和卡牌"""
        rules = self.search(query)
        cards = self.card_service.search_cards_by_name(query, limit=3)

        return {
            "rules": rules,
            "cards": cards
        }
```
