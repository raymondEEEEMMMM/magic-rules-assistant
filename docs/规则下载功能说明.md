# 万智牌规则自动下载功能说明

## 📋 功能概述

实现了从官网自动下载和更新万智牌完整规则的功能，支持：
- ✅ 自动下载最新规则（TXT格式）
- ✅ 规则版本检查和增量更新
- ✅ 规则内容解析（规则条目、关键词异能、术语表）
- ✅ 定时任务调度
- ✅ 向量化数据准备

## 🎯 格式选择

**推荐使用 TXT 格式**

### 对比分析

| 格式 | 向量化友好度 | 提取难度 | 结构保留 | 推荐度 |
|------|-------------|---------|---------|--------|
| **TXT** | ⭐⭐⭐⭐⭐ | 简单 | 一般 | ✅ **推荐** |
| DOCX  | ⭐⭐⭐⭐ | 中等 | 好 | ✅ 可用 |
| PDF   | ⭐⭐ | 困难 | 好 | ❌ 不推荐 |

### 选择 TXT 的原因

1. ✅ **纯文本，向量化最友好** - 直接分词即可，无需复杂解析
2. ✅ **解析最简单** - 直接读取，正则提取规则编号和内容
3. ✅ **结构化保留** - 规则编号层次清晰，便于后续处理
4. ✅ **处理速度快** - 无需额外解析库，内存占用小
5. ✅ **文件体积小** - 约 943 KB，易于存储和传输
6. ✅ **易于更新** - 直接覆盖下载，无需复杂格式转换

## 🚀 使用方法

### 1. 手动更新规则

```bash
# 方式一：使用更新脚本
python3 update_rules.py

# 方式二：使用测试脚本
python3 test_rule_downloader.py

# 方式三：通过 API（服务运行时）
curl -X POST "http://localhost:8000/api/rules/update"
```

### 2. 自动定时更新

服务启动后会自动启动定时任务：
- **每天凌晨 3:00**: 检查规则更新
- **每周一上午 10:00**: 全面检查规则

### 3. API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/rules/update` | POST | 更新规则（可选 force 参数） |
| `/api/rules/parse` | GET | 解析规则文件 |
| `/api/rules/vectorization` | GET | 获取向量化数据 |
| `/api/rules/status` | GET | 获取规则状态 |

## 🔧 核心功能

### 1. 规则下载（RuleDownloader）

```python
from services.rule_downloader import RuleDownloader

downloader = RuleDownloader()

# 下载规则
result = downloader.download_rules()

# 解析规则
parse_result = downloader.parse_rules()

# 获取向量化数据
vector_data = downloader.get_rules_for_vectorization()
```

### 2. 定时任务（RuleScheduler）

```python
from services.scheduler import rule_scheduler

# 启动调度器
rule_scheduler.start()

# 手动触发更新
result = await rule_scheduler.update_now()

# 获取状态
status = rule_scheduler.get_status()
```
