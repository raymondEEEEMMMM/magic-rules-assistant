## 脚本说明

### deployment/scripts/start.sh

本地开发环境启动脚本，功能：
- 自动激活虚拟环境
- 检查并创建数据目录
- 初始化数据库（如果需要）
- 启动 API 服务

使用方式：
```bash
bash deployment/scripts/start.sh
```

### deployment/scripts/cloudbaserc.json

CloudBase 配置文件，包含：
- 环境配置（envId）
- 函数配置（名称、描述、运行时、处理器、超时、内存）
- 环境变量（ENVIRONMENT、WECHAT_TOKEN）
- HTTP 访问路径（path）

### deployment/docker/Dockerfile

Docker 镜像构建文件：
- 基于 Python 3.10 slim 镜像
- 安装项目依赖
- 复制代码和数据
- 配置运行参数（端口 80）
- 使用 uvicorn 启动服务

使用方式：
```bash
docker build -f deployment/docker/Dockerfile -t mtg-rules-api .
docker run -p 80:80 mtg-rules-api
```
