# 公众号UI按钮设计文档

## 设计概述

本文档定义了万智牌规则问答公众号的自定义菜单按钮UI设计方案，遵循微信小程序公众号的交互规范和用户体验最佳实践。

---

## 一、主菜单结构

### 一级菜单（3个）

| 位置 | 按钮名称 | 功能描述 | 页面类型 |
|------|---------|---------|---------|
| 左侧 | 📜 规则查询 | 规则查询入口 | 二级菜单 |
| 中间 | 🎴 卡牌搜索 | 卡牌查询入口 | 二级菜单 |
| 右侧 | ❓ 帮助中心 | 帮助和设置 | 二级菜单 |

---

## 二、二级菜单详细设计

### 2.1 左侧菜单：📜 规则查询

#### 按钮配置
```
┌─────────────────────────────┐
│  📜 规则查询                  │
├─────────────────────────────┤
│  🔍 关键词搜索                │ → 弹出搜索框
│  📚 常见规则                  │ → 列表页面
│  ⚡ 快速查询                  │ → 常用关键词标签页
│  🏷️ 异能词典                  │ → 216个关键词分类浏览
└─────────────────────────────┘
```

#### 按钮详情

| 二级按钮 | 图标 | 功能 | 跳转方式 |
|---------|------|------|---------|
| 🔍 关键词搜索 | 搜索图标 | 搜索关键词异能 | 点击弹出输入框 |
| 📚 常见规则 | 书本图标 | 浏览热门规则（飞行、践踏等） | 跳转到规则列表页 |
| ⚡ 快速查询 | 闪电图标 | 显示高频查询标签 | 跳转到标签云页面 |
| 🏷️ 异能词典 | 标签图标 | 216个关键词分类浏览 | 跳转到分类页面 |

#### 交互流程
1. 用户点击"🔍 关键词搜索" → 弹出搜索框
2. 输入关键词 → 显示匹配规则
3. 点击结果 → 展示规则详情（卡片式展示）

## 四、交互设计规范

### 4.1 按钮尺寸规范

| 按钮类型 | 宽度 | 高度 | 圆角 | 字体大小 |
|---------|------|------|------|---------|
| 主按钮 | 100% | 44px | 6px | 16px |
| 次按钮 | 48% | 40px | 4px | 14px |
| 图标按钮 | 36px | 36px | 50% | - |
| 标签按钮 | auto | 32px | 16px | 13px |

### 4.2 颜色规范

| 用途 | 背景色 | 文字色 | 边框色 |
|------|--------|--------|--------|
| 主按钮 | #07C160（微信绿） | #FFFFFF | #07C160 |
| 次按钮 | #F5F5F5 | #353535 | #E5E5E5 |
| 危险按钮 | #FA5151 | #FFFFFF | #FA5151 |
| 禁用状态 | #F5F5F5 | #C9C9C9 | #E5E5E5 |

### 4.3 间距规范

- 菜单项垂直间距: 12px
- 按钮组水平间距: 8px
- 内容内边距: 16px
- 卡片外边距: 12px

### 9.2 后端事件处理

在 `backend/routes.py` 中添加菜单事件处理：

```python
@router.post("/wechat")
async def handle_wechat_message(request: Request):
    # 解析消息
    xml_data = await request.body()
    msg = parse_xml(xml_data)

    # 处理菜单点击事件
    if msg['MsgType'] == 'event':
        if msg['Event'] == 'CLICK':
            event_key = msg['EventKey']
            return handle_menu_click(event_key, msg)

    return XMLResponse(content=generate_response(msg))

def handle_menu_click(event_key, msg):
    """处理菜单点击事件"""
    handlers = {
        'RULE_SEARCH': '请输入关键词搜索规则，例如：飞行',
        'COMMON_RULES': generate_common_rules(),
        'QUICK_QUERY': generate_quick_query(),
        'KEYWORD_DICT': generate_keyword_dict(),
        'CARD_SEARCH': '请输入卡牌名称，例如：黑莲花',
        'CARD_FILTER': generate_filter_options(),
        'RANDOM_CARD': get_random_card(),
        'POPULAR_CARDS': generate_popular_cards(),
        'USER_GUIDE': generate_user_guide(),
        'CONTACT_US': generate_contact_info(),
        'SETTINGS': generate_settings(),
        'STATISTICS': generate_statistics(),
    }

    return XMLResponse(content=generate_text_response(
        msg,
        handlers.get(event_key, '功能开发中...')
    ))
```
