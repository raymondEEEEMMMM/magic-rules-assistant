# 万智牌规则问答 - 快速参考

## 🚀 快速启动

```bash
# 1. 安装依赖
pip3 install -r requirements.txt

# 2. 初始化数据库
python3 backend/init_data.py

# 3. 更新规则（可选）
python3 update_rules.py

# 4. 启动服务
./start.sh
```

## 📡 API 端点

### 基础
- `GET /` - 服务状态
- `GET /docs` - API文档

### 查询
- `GET /api/search?q=` - 搜索规则
- `GET /api/keyword?k=` - 查询异能
- `GET /api/card?n=` - 查询卡牌

### 规则管理
- `POST /api/rules/update` - 更新规则
- `GET /api/rules/parse` - 解析规则
- `GET /api/rules/vectorization` - 向量化数据
- `GET /api/rules/status` - 规则状态

## 📝 注意事项

1. 规则文件格式为 TXT（便于向量化）
2. 定时任务：每天03:00、每周一10:00
3. 生产环境建议使用 PostgreSQL
4. 需要配置微信参数才能接入公众号

## 🆘 故障排查

| 问题 | 解决方案 |
|------|---------|
| 数据库错误 | 检查 `DATABASE_PATH` 路径 |
| 规则下载失败 | 检查网络连接和下载地址 |
| 微信验证失败 | 检查 Token 和 EncodingAESKey |
| 依赖安装失败 | 使用 pip3 安装 |
