# 微信集成快速开始

## 前置条件

1. 已部署云函数 `magic-rules-api`
2. 已配置微信公众号

## 步骤 1: 配置环境变量

```bash
# 进入集成目录
cd wechat_integration

# 复制配置模板
cp config/env.example config/.env

# 编辑配置（填入 AppID 和 AppSecret）
vim config/.env
```

## 步骤 2: 验证配置

```bash
python3 scripts/verify_wechat.py
```

## 步骤 3: 设置菜单

```bash
python3 scripts/set_menu.py
```

## 步骤 4: 在微信后台配置

如果步骤 3 失败（订阅号无权限），需要手动配置：

1. 登录 https://mp.weixin.qq.com/
2. 自定义菜单 → 创建菜单
3. 参考 `config/menu.json` 配置

## 测试

在微信中发送消息测试：
- 发送 `飞行` → 返回关键词解释
- 发送 `战斗` → 返回规则搜索结果
- 发送 `卡牌:闪电击` → 返回卡牌信息
